Import Rules & Guidelines
1
Required columns: Date, Depot, State, SR Ref, Party, Qty, Value — row 1 must be headers.
2
SR Ref is the sales return reference number. Can be alphanumeric (e.g. SR-2025-001).
3
Qty must be a whole number. Value must be numeric without ₹ symbol or commas.
4
Date: use YYYY-MM-DD or Excel date serial format for best compatibility.
5
Depot short codes: HO, CBE, DIRECT are auto-mapped to full names.
6
Use the Template button for a pre-formatted file with all column headers ready.
